Frizerski Salon BeŠk

Opis
Informacioni sistem za online zakazivanje termina kod frizera.

 Tehnologije
- ASP.NET Core 8.0
- Entity Framework Core
- Microsoft SQL Server
- Bootstrap 5

Funkcionalnosti

Klijent
- Registracija i prijava
- Zakazivanje termina
- Pregled mojih termina
- Otkazivanje termina

Admin
- Upravljanje frizerima
- Upravljanje uslugama
- Pregled svih termina

Pokretanje

Lokalno
```bash
dotnet run
```

Docker
```bash
docker-compose up --build
```

Aplikacija: http://localhost:5000

Admin prijava:
admin@besk.ba / Admin123!
